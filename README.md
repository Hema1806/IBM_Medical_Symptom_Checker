# 🩺 AI-Powered Medical Symptom Checker & First Aid Assistant using IBM watsonx Orchestrate

## 📌 Project Overview

The AI-Powered Medical Symptom Checker is an intelligent conversational assistant developed using IBM watsonx Orchestrate. It provides users with general symptom guidance, first aid recommendations, emergency awareness, and healthy lifestyle suggestions through natural language interactions.

---

## 🎯 Problem Statement

Many individuals require reliable preliminary healthcare guidance before consulting a medical professional. This project aims to provide general symptom information and first aid recommendations while encouraging users to seek professional medical care when necessary.

---

## 🎯 Objectives

- Develop an AI-powered Medical Symptom Checker.
- Provide symptom guidance.
- Suggest first aid.
- Promote healthy lifestyle habits.
- Identify emergency warning signs.
- Utilize a medical knowledge base.

---

## 🚀 Features

- Symptom guidance
- First aid recommendations
- Emergency warning awareness
- Healthy lifestyle suggestions
- Conversational AI
- Knowledge-based responses

---

## 🛠 Technologies Used

- IBM Cloud

---

## 📚 Knowledge Base

- Common Symptoms Guide
- First Aid Guide
- Emergency Warning Signs
- Healthy Lifestyle Guide

---

## ⚙️ Workflow

1. User enters symptoms.
2. AI understands the query.
3. Knowledge Base is consulted.
4. IBM Foundation Model generates guidance.
5. User receives health recommendations.
